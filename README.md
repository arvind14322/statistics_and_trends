# Statistics and Trends Assignment

## Project Overview

This repository contains my implementation of the *Statistics and Trends* assignment based on the provided template. It focuses on applying descriptive statistics and simple trend analysis to a real-world dataset, using Python for data loading, summarisation, and basic exploratory analysis.

## Repository Contents

- `statistics_and_trends.py` – Main Python script that loads the dataset, computes summary statistics, and performs the required analyses.  
- `data.csv` – Input dataset used by the script for the assignment tasks.  
- `requirements.txt` – List of Python dependencies needed to run the script.  
- `.gitignore` – Standard ignore rules for Python projects.  

## Assignment Objectives

The assignment is designed to:

- Demonstrate the use of Python for basic statistical analysis.  
- Compute key descriptive statistics (such as measures of central tendency and spread) from a tabular dataset.  
- Explore simple trends or patterns in the data using programmatic analysis.  
- Produce clear, reproducible code that follows the given template structure.  

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/arvind14322/statistics_and_trends.git
   cd statistics_and_trends
   ```
2. Create and activate a virtual environment (optional but recommended).  
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the analysis script:
   ```bash
   python statistics_and_trends.py
   ```

## Usage Notes

- Ensure that `data.csv` is present in the repository root and matches the expected format used in `statistics_and_trends.py`.  
- You can modify or extend the script to add additional visualizations or statistical measures as needed for further analysis.
